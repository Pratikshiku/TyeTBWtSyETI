### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 个人中心
4. 历史竞拍历史
5. 竞拍订单管理
6. 留言板管理
7. 首页
8. 商品拍卖
9. 竞拍公告
10. 留言反馈
11. 个人中心
12. 后台管理


**管理员功能：**
1. 登录
2. 个人中心
3. 用户管理
4. 商品类型管理
5. 拍卖商品管理
6. 历史竞拍管理
7. 留言板管理
8. 轮播图管理
9. 竞拍公告
